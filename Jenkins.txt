Parameterize Jenkins Jobs.
- Go to configure
- Tick this project is paramenterize 
- select choice parameter
- Select parameter name and value eg. Browser : Firefox, Chrome, Edge
- save (Build will change to build with parameters)
- Go to configure and modify -> In maven command, change "$Variable-name"
eg test-PRegression -Dbrowser="$Browser"
similarly more variables can be configured

Schedule Jenkins Jobs. (Keeeping the servers on)
