GIT : Vrsion control system, it is a software on system
Git bash recommended for windows and terminal for mac
Github : website that tracks, remote repository and controls Repo.

Steps : 
1. Create an account on Github and Repo
2. Install git, configure : git config : global user.name "Name" 
                                         : git config : global user.email "email"
    To check : git config --list
3. After new files or modified : git add <file name> or git add . for all files.
4. git commit -m "message"
5. git push origin main (pushes files to remote repo)
Note : always pull latest code before editing.	

Work Flow : Git repo -> clone -> Changes -> add -> commit -> push

Clone : clone remote repo to local : git clone <url>
cd : change directory 
ls -a : check if the folder is tracked by git (.git file appears)	
git status : display the status of code
                 : Modified file : modified but not updated (changed)
				 : Untracked : No info to git (new file)
				 : Staged : Ready to commit
				 
Branch command : 
git branch : to check branch
git branch -m name : rename a branch 
git checkout branchname : navigate to branch 
git checkout -b newBranchName : create a new branch				
git branch -d branchname : delete a branch (cannot delete selected branch)

Merging code : 
1. Using Git 
     - git diff branchnName : to compare
     - git merge branchName	 
	 
2. Using Github
     - Create a PR (pull request)
                 PR : Tell others about changes you've pushed to a branch in a repo on github.
	 - On github -. compare and pull request
     - Create a pull request
     - Merge pull request
     - Confirm merge
